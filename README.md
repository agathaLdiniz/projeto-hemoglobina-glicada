# Análise de Dados - Hemoglobina Glicada

## 📊 Sobre o Projeto

Este projeto consiste em uma análise exploratória de dados de pacientes com hemoglobina glicada (HbA1c) acima de 6,5%.

## 🎯 Objetivos

- [ ] Explorar e entender os dados de pacientes
- [ ] Identificar padrões e correlações
- [ ] Visualizar informações importantes
- [ ] Gerar insights para ação
